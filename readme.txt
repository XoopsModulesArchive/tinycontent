 Install Instructions:
 =====================

 - Copy tinycontent in your modules folder  
 - Install it via the modules admin menu

 Usage Infos:
 ============

 If you click on the admin Logo the first
 time, you see the following links:

 - Add Content
 - Add PageWrap
 - Edit/Delete Content

 1. Add Content
    -----------
   -You can add your content like you
    write your News.
   -The Linktitle will be displayed in
    the Tinycontent block, but only if
    you have activated the "visible"
    checkbox
   -If you want to have a submenu item
    in the xoops mainmenu activate the
    submenu box
   -Deactivate linebreak conversion if
    you put html code in the textfield
    This will prevent white spaces in
    front of the html
 2. Add PageWrap
    ------------
   -Here you can include html-files
    in your Xoops.
 3. Edit/Delete Content
    -------------------
   -Here you can define which Content
    should be displayed when you click
    on the TinyContent link in the
    mainmenu.
   -You can also change the Link-ID. This
    controls the order of the links in
    the TinyContent block.
 4. Preferences
    -----------
   -You can control the comment settings
   -You can choose between Xoops Editor
    and the Wysiwyg Editor
    (Needs Internet Explorer 5.5 or above!)
